# 团队 BookMarks

## Install

依赖 `^@node8.0`

```js
npm install
```

## Build

```js
npm run build
```

## Output

`/dist`

## Edit

*   `~ cd /bookmarks`
*   `vi XXXX.json`, xxxx 为目录集

```json
{
    "name": "显示的名字",
    "url": "跳转的URL",
    "icon": "图标" // http://图标url || fa-icon@https://fontawesome.com/icons || ''
}
```
